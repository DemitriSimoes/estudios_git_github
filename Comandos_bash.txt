
Comandos do Bash

ls -> lista arquivos
cd -> move entre diretorios
pwd -> mostra diretorio atual
cat -> mostra o conteudo do arquivo atraves do prompt
rm -> remove arquivo ou diretorio

Comandos do Git

git init . -> inicializa o repositorio git vazio (cria uma pasta .git/ no diretorio atual)
git config --global --add safe.directory D:/Analyst/Git_GitHub (caminho para dar permissão de acesso, pois o git normalmente verifica quando não tem permissão ou quando está em uma partição diferente, nesse caso D:)
git status -> me mostra como está o meu diretorio
git add .(para pasta atual) ou <nomedo arquivo> -> adiciona os arquivos que eu quero que vá para o commit
git commit -m "<mensagem>" -> cria um save um checkpoint do arquivo, commit feito, alterações salvas
git config --global user.name <nome do usuario> -> adiciona o email que vai usar o git e se digitar sem o email só o comando mostra qual email ou usuario que esta usando
git config --global user.email <nome do email>
git log -> verifica todo o log de commit feito
